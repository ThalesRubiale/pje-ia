Criar sua chave da API da Anthropic

A extensão funciona com os modelos Claude (Anthropic) e Gemini (Google) — basta ter a chave do provedor do modelo escolhido. Esta seção cobre a Anthropic; a chave do Google está logo abaixo. Você precisa de uma chave de API própria e de crédito na sua conta. Leva ~3 minutos.

1

Acesse o Console da Anthropic

Crie uma conta (ou entre) no console de desenvolvedores. É diferente do chat claude.ai.

Abrir console.anthropic.com ↗
2

Adicione crédito (botão verde)

Vá em Settings → Billing (Plans & Billing) e clique no botão verde para adicionar crédito pré-pago. Um valor pequeno já basta para começar.

console.anthropic.com/settings/billing
+ Add credits ← clique aqui
Abrir Billing ↗

Nesta etapa você decide o valor — a extensão não cobra nada e não adiciona crédito por você.

3

Crie a API Key

Vá em Settings → API Keys e clique em Create Key. Dê um nome (ex.: "PJe IA").

console.anthropic.com/settings/keys
Create Key ← clique aqui
Abrir API Keys ↗
4

Copie e cole na extensão

A chave aparece uma única vez (começa com sk-ant-...). Copie na hora.

  • Clique no ícone ⚖️ PJe IA na barra do Chrome.
  • Cole a chave no campo, escolha o modelo e clique em Salvar.

Criar sua chave da API do Google (Gemini)

Para usar os modelos Gemini 3.6 Flash e Gemini 3.5 Flash-Lite, crie uma chave no Google AI Studio. Leva ~2 minutos.

1

Acesse o Google AI Studio

Entre com sua conta Google e abra a página de chaves de API.

Abrir aistudio.google.com/apikey ↗
2

Crie a API Key

Clique em Create API key (ou "Criar chave de API"). A chave começa com AIza....

Há um nível gratuito com cota limitada — bom para testar. Para uso regular (e para a busca de jurisprudência com Google Search), ative o faturamento do projeto no Google Cloud, pela própria tela do AI Studio.

3

Cole na extensão

  • Clique no ícone ⚖️ PJe IA na barra do Chrome.
  • Cole a chave no campo "Chave da API — Google (modelos Gemini)", escolha um modelo Gemini e clique em Salvar.

Diferenças nos modelos Gemini: as citações de página aparecem no próprio texto da resposta (sem os marcadores [n] clicáveis) e a geração de arquivo .docx fica indisponível — esses dois recursos usam a API da Anthropic.

Qual modelo escolher?

O selo na barra de ferramentas do chat mostra sempre o modelo e o nível de raciocínio ativos. Preços em US$ por 1 milhão de tokens (entrada / saída) — o cache automático reduz a entrada repetida em ~90% nos turnos seguintes da mesma conversa.

ModeloJanela / PDFPreçoPerfil
Claude Haiku 4.5 (padrão)200 mil tokens / 100 págs.1 / 5Rápido e barato; todos os recursos (citações [n], .docx). Melhor ponto de partida.
Claude Sonnet 51M tokens / 600 págs.3 / 15Autos volumosos com todos os recursos; mais lento e caro que o Haiku.
Claude Opus 4.81M tokens / 600 págs.5 / 25Qualidade superior para análises delicadas.
Claude Fable 51M tokens / 600 págs.10 / 50O mais capaz — e o mais caro e lento. Para casos excepcionais.
Gemini 3.6 Flash1M tokens / 1000 págs.1,50 / 7,50Rápido, multimodal e com ótimo custo para autos grandes. Citações no texto; sem .docx.
Gemini 3.5 Flash-Lite1M tokens / 1000 págs.0,30 / 2,50O mais barato e veloz de todos — ideal para triagens e resumos rápidos. Citações no texto; sem .docx.

Regra prática: Haiku para o dia a dia; Gemini 3.5 Flash-Lite quando o custo importa mais; Sonnet 5 ou Gemini 3.6 Flash para autos volumosos; Opus/Fable quando a qualidade da análise vale o preço.

Como usar bem o chat (fluxo recomendado)

  1. Abra os autos no PJe e clique em ⚖️ Analisar com IA.
  2. Clique em ⟳ Carregar todas as peças (abaixo da lista): o PJe só carrega os documentos conforme a linha do tempo é rolada — sem esse passo, a lista pode estar incompleta.
  3. Marque as peças relevantes — o atalho principais seleciona de uma vez as destacadas por categoria (decisões, audiências, petições, provas); a busca e o @ ajudam a achar peças pelo nome.
  4. Pergunte. Ligue 🔍 Jurisprudência se quiser fundamentação em fontes oficiais; use 📄 Gerar .docx (modelos Claude) para relatórios em Word.
  5. Siga a conversa: adicionar peças no meio é barato; para remover várias ou mudar de assunto, prefira ⟲ Nova conversa.
  6. Personalize o assistente: nas opções, o campo "Instruções personalizadas" diz ao modelo quem você é e como quer as respostas (ex.: "Sou assessor de vara criminal; destaque prazos prescricionais") — vale para os modelos Claude e Gemini, no chat e no .docx.
  • Comece marcando só as peças relevantes para a pergunta — cada peça selecionada entra no contexto e é reenviada (com desconto de cache de ~90%) a cada turno.
  • Adicionar peças no meio da conversa é barato e seguro: a peça nova entra uma única vez, ao final, e o restante da conversa aproveita o cache. Não precisa criar um novo chat para adicionar documentos.
  • Desmarcar peça libera contexto na hora, mas invalida o cache daquele ponto em diante (o turno seguinte re-paga o envio das peças restantes). Use quando o medidor encher; para trocar de assunto ou remodelar a seleção inteira, prefira ⟲ Nova conversa.
  • O medidor no rodapé é o guia: perto de 90% da janela do modelo, desmarque peças ou abra uma conversa nova. Vale para Claude e Gemini (ambos têm cache automático).
🔒 Suas chaves ficam só neste navegador (não são enviadas a lugar nenhum além das próprias APIs da Anthropic e do Google). As peças que você marcar são enviadas ao modelo escolhido para análise — processos podem conter dados sigilosos, use com critério. Detalhes na política de privacidade.
